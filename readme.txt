shortcode to include the BFV Widget within wordpress

== Description ==

Sorry, no english descriptions as this plugin is adressed to german (bavarian) user only.
Dieses Plugin stellt einen Shortcode zur Verfügung, der hilft das Widget des Ergebnisdienst des Bayerischen Fussball Verbandes zu nutzen.

== Installation ==

Das Plugin herunter geladen und wie üblich im Blog aktivieren.
Die Homepage beim BFV für die Benutzung des Widgets freischalten. (siehe Anleitung BFV)
Den Shortcode (s.u.) in die gewünschten Seiten oder Beiträge einbauen
